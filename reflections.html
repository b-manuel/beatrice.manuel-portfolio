Resume Internship Projects Reflections

Technical Journey & Growth

Reflecting on key projects from University to Enterprise Internship.

Internship CAC Project

The Impact of Automation

The CAC Automation project was a pivotal moment in my internship. We were tasked with replacing a costly manual audit process. Evaluating solutions like AppSheet versus Python Streamlit taught me that development isn't just about code—it's about balancing cost, speed, and security.

Seeing the turnaround time drop from 1 hour to under 15 minutes demonstrated the tangible business value of the systems we build.

System Design

Connecting Frontend to Backend

Mapping the eSIM User Journey challenged me to look beyond the user interface. I had to visualize how a simple tap in the GlobeOne app triggers complex interactions across backend systems like NG1, CXS, and RAVEN.

It highlighted the importance of error handling—planning for what happens when a QR code fails or a payment gateway times out.

Data Analytics

Mastering Data Storytelling

My university projects, specifically the Global Sales Analytics and Population Dashboards, pushed me to learn DAX and M languages deeply.

I learned that data is only as useful as its presentation. Creating interactive visualizations for COGS and revenue allowed me to transform raw numbers into actionable business insights.

Software Dev

Building Full-Stack Solutions

Developing the Digitalized Pharmaceutical Management System and the Clinic Queue Console using C# and SQL gave me my first taste of full-stack logic.

These projects taught me how to manage databases effectively using SQL Server and how to structure applications using the MVC pattern to ensure scalability.
